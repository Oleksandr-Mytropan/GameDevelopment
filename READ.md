# 🎮 Unity 3D Game Development Project

## 📌 About the project
This is a personal 3D game development project created in Unity.  
The goal of this project is to practice game development, physics, animations, and overall game architecture.

## 🛠️ Technologies used
- Unity
- C#
- Git
- Blender (for environment assets)

## 🎮 Features
- Player movement
- Camera system
- Animations
- Basic collision system
- Simple 3D environment

## 🚀 Project status
This project is currently in development and being improved step by step.

## 📂 How to run
1. Clone the repository
2. Open the project in Unity
3. Run the scene

## 📚 What I learned
- Basics of Unity engine
- Working with animations
- Handling collisions and physics
- Using Git for version control
